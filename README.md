# NestFinder PG-Life

NestFinder PG-Life is a robust, full-stack PG portal designed to streamline the property searching and management experience for both tenants and owners.

## 🚀 Key Features

*   **AI-Powered Chatbot:** Integrated with **ChatGPT** to provide instant, automated assistance to users.
*   **Secure Access:** Full platform features, including the expense tracker, are restricted to registered users. Login is mandatory for a personalized experience.
*   **Expense Splitter:** A dedicated utility for tenants to manage and split room expenses and utility bills effortlessly.
*   **Live Distance Calculator:** Real-time distance tracking between PG locations and key landmarks.
*   **Dynamic Map Views:** Interactive and responsive maps to locate properties with ease.
*   **Owner Dashboard:** A specialized interface for PG owners to manage listings and tenant details. (Accessible via the dedicated link in the footer).

## 🛠️ Tech Stack

*   **Frontend:** React.js
*   **Backend:** PHP
*   **Database:** MySQL

## 📋 How to Run

1. Clone this repository.
2. Ensure your local server (XAMPP/WAMP) is running with MySQL.
3. Import the database file into `phpMyAdmin`.
4. Navigate to the project directory and run the React frontend using `npm start`.
